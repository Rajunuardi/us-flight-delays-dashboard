# Dashboard Keterlambatan Penerbangan AS

Project ini menganalisis keterlambatan penerbangan di Amerika Serikat menggunakan data publik dari Kaggle.
Proses analisis mencakup pembersihan data menggunakan Excel dan visualisasi data menggunakan Tableau
dalam bentuk dashboard interaktif.

## Tools
- Microsoft Excel (Data Cleaning)
- Tableau Public (Dashboard & Visualisasi)

## Dataset
Sumber data: Kaggle – US Flight Delays (2017–2022)

## Struktur Folder
- Data/raw/ : data mentah hasil unduhan dari Kaggle
- Data/cleaned/ : data yang sudah dibersihkan dan digunakan untuk dashboard
- Data/tableau/ : file dashboard Tableau (.twbx)
- Data/images/ : gambar/screenshot dashboard

## Dashboard
Link Tableau Public:  
https://public.tableau.com/app/profile/raju.nuardi.akbar/viz/USFlightDelaysDashboard20172022/Dashboard1

## Insight Singkat
- Faktor maskapai (Carrier Delay) menjadi salah satu penyumbang terbesar keterlambatan penerbangan.
- Pola keterlambatan menunjukkan variasi musiman dan perubahan tren dari tahun ke tahun.
